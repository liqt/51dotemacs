#+TITLE: 项目说明

这个项目存放我的emacs的配置文件。

* Windows平台下快速搭建Emacs

  1. 到 [[https://code.google.com/p/emacs-for-windows][这里]] 下载编译好的emacs，将其解压到C:/Emacs目录下
  2. 下载本项目的配置文件master.zip，将 *51dotemacs-master* 的内容解压到C:/Emacs目录下
  3. 将C:/Emacs/.emacs.d/elisp目录下的XXX.example.el该名为XXX.el，并酌情修改，主要是设置环境变量
  4. 导入quick_open.reg和set_path.reg到注册表，前者是右键打开文件，后者是设置Emacs的HOME目录
  5. 下载微软雅黑consolas字体库(YaHei.Consolas.1.11b.ttf)，并将其导入windows字体库。
  6. 查看并修改C:/Emacs/.emacs文件，将Emacs变成你要的样子

* Linux平台快速搭建Emacs

** 安装emacs
这里推荐使用emacs23以上版本，因为低于emacs23对utf-8的支持不是特别好。

*** emacs23
这个是gtk版本，也就是用了gtk的ui库。
: apt-get install emacs23

*** emacs23-lucid
这个是x-window版本，ui库比其gtk简单些吧。
: apt-get install emacs23-lucid

*** emacs23-nox
这个版本是不支持x-window的，服务器可以选择装这个。
: apt-get install emacs23-nox

** emacs配置文件
安顺序执行以下的指令，就可以使用我的emacs配置。
: root@ubuntu:~# cd ~/
: root@ubuntu:~# git clone git://github.com/wuyao721/51dotemacs.git
: root@ubuntu:~# cp 51dotemacs/.screenrc ~/
: root@ubuntu:~# cp 51dotemacs/.emacs ~/
: root@ubuntu:~# cp -r 51dotemacs/.emacs.d ~/

** .screenrc
服务器版的emacs还须配上一个screen（可以参考我这篇文章――[[http://www.wuyao721.com/emacs-screen.html][一个emacser的screen]] ），把 *.screenrc* 也拷到HOME目录。
: cp 51dotemacs/.screenrc ~/
